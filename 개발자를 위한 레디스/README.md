# 이 책에서 다루는 내용
- 마이크로서비스 아키텍처에서 NoSQL 사용
- 레디스 설치, 환경 구성 및 기본 설정 파일 안내
- 레디스의 다양한 자료 구조 활용법
- 효율적인 키 관리 방법
- 레디스를 이용해 성능을 향상시킬 수 있는 다양한 설계 사용 사례
- 레디스를 캐시 및 세션 스토어로 활용하는 방법
- 레디스를 메시지 브로커로 활용하는 pub/sub 및 stream 사용법
- 레디스 데이터의 영구 저장 방법
- 고가용성 설정을 위한 복제 구조 및 센티널 활용
- 확장성을 갖춘 클러스터 사용 방법
- 클라이언트 핸들링과 보안
- 레디스 모니터링 구성 및 버전 업그레이드 방법
